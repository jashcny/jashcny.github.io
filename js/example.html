Product filter for sharing economy platform

Content of the website changes on daily basis, to let user to be able to see new content everyday and at the same time they can browse through the rest of the content easier. With this filter, the problem solved!

User can view the type of food they interested, or if they have limited budget, or if they want to see high rating services or goodies, they can get it sorted out via this filter.
